# JAVASCRIPT_BATCH_18_ASSIGNMENTS_6_LOGIN_SIGNIN

## Project Description
This project is a simple, fully responsive Login and SignUp system using HTML, CSS, and vanilla JavaScript. It allows users to register, login, and access a home page. Features include:

- Login and SignUp forms with validation
- Success alerts for login and account creation
- Redirect to home page after login/signup
- Logout functionality
- Dark/Light mode toggle
- Animated buttons and input fields
- Fully responsive design

## Files
- `index.html` - Home page with logout button
- `signIn.html` - Login page
- `signUp.html` - SignUp page
- `style.css` - All styles including animations and dark/light mode
- `script.js` - JavaScript for login, signup, logout, alerts, and dark/light toggle

## How to Use
1. Open `signIn.html` or `signUp.html` in a browser.
2. Register using SignUp or login with existing credentials.
3. On successful login/signup, an alert appears, and clicking OK redirects to `index.html`.
4. Click the logout button to return to the login page.

## Features
- Simple and clean UI
- Alerts for user actions
- Home page access after authentication
- Dark/Light mode toggle
- Animations for buttons and inputs
